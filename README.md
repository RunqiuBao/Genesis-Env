# Genesis Env
[https://github.com/Genesis-Embodied-AI/Genesis.git](https://github.com/Genesis-Embodied-AI/Genesis.git)

## Description
Having a simulation and digital twin environment is a must for any robotics engineer.
- Build an env with Genesis simulator by a single make command `make docker`.

## How to use
In a docker container:
```
git clone https://github.com/RunqiuBao/config_genesis_in_docker.git
cd config_genesis_in_docker/
make docker
```
